Software

R packages

  • effectbounds: non-overlap bounds for causal effects. [github]. Author.
  • TargetedRisk: Targeted estimation of risk standardization parameters. [github]. Author.
  • AdaptiveConformal: Adaptive Conformal Inference (ACI) algorithms. [github]. Author.
  • RNHANES: Tools for downloading and analyzing CDC NHANES data, with a focus on analytical laboratory data. [github] [CRAN]. Original author; now managed by Silent Spring Institute.
  • Reactor: Reactive notebooks for R. [website] [github] [lightning talk video]. Author.

Julia packages

  • TargetedMSM.jl: Targeted Bayesian inference for marginal structural models, using automatic differentiation to support arbitrary working models and loss functions. [github]. Author.
